# TeamBanks - Site de Consultoria Fitness

Site institucional da consultoria fitness do Treinador Banks.

## Estrutura

```
/Banks
├── index.html      # Página principal
├── style.css       # Estilos customizados
├── script.js       # Scripts e interações
├── media/          # Imagens e vídeos
│   ├── resultados/ # Imagens de resultados
│   └── *.png, *.mp4
└── README.md
```

## Tecnologias

- Tailwind CSS (via CDN)
- Font Awesome 6
- AOS Animate On Scroll
- Plus Jakarta Sans (Google Fonts)

## Como usar

1. Abra `index.html` em um navegador
2. Para desenvolvimento local, use um servidor como Live Server

## Otimizações

- Animações otimizadas para 60fps
- Lazy loading de imagens
- Scroll performático com passive listeners
- Suporte a prefers-reduced-motion
